#  Plant Disease Detection System

An AI-powered plant disease detection system built using deep learning, designed to classify plant leaf diseases from images with high accuracy. The system follows a full-stack architecture with a FastAPI backend and a React TypeScript frontend.

## ⚙️ How It Works

1. User uploads a plant leaf image
2. Image is preprocessed (resize, normalize)
3. CNN model predicts disease class
4. System returns prediction with confidence score

---

## 📊 Model Details

* Model Type: Convolutional Neural Network (CNN)
* Classes: 30+ plant diseases
* Input: Leaf images
* Output: Disease label + confidence score
